# 시맆 리워드 패스 모노레포

이 저장소는 Spring Boot 기반 백엔드와 React 기반 프런트엔드를 각각 `backend`, `frontend` 디렉터리로 분리하고, Docker Compose로 한 번에 실행할 수 있도록 구성되었습니다.

## 실행 방법

1. Docker와 Docker Compose가 설치되어 있어야 합니다.
2. 저장소 루트에서 다음 명령을 실행합니다.

```bash
docker compose up --build
```

3. 서비스가 기동되면 다음 주소에서 접속할 수 있습니다.
   - 프런트엔드: http://localhost:5173
   - 백엔드 API: http://localhost:8080

프런트엔드는 `VITE_API_BASE_URL` 환경 변수를 통해 백엔드 주소를 참조하며, Compose 환경에서는 자동으로 백엔드 컨테이너를 바라보도록 설정되어 있습니다.

✨ 주요 기능 및 차별점
1. Portal 기반 신원 인증 (Trusted Auth)

학교 공식 포털(portal.uos.ac.kr)에 직접 로그인하는 구조로 학생 여부를 검증합니다.

Playwright 기반 크롤링 인증

재학생만 참여 가능

1인 1매 원칙 보장

외부인 수령 불가

2. Redis 대기열 기반 선착순 처리 (Fair Concurrency)

인기 행사 쿠폰 오픈 시 발생하는 트래픽 폭주 상황에 대응하기 위해 Redis 기반의 대기열 시스템을 구현했습니다.

요청 순서별 큐 발행

DB 부담 최소화

서버 부하 증가 상황에서도 안정적 동작

공정한 발급 순서 보장

3. AI 기반 리워드 최적 금액 추천 (AI Optimization)

LightGBM 모델을 통해 행사 특성·기간·대상 등을 분석하여 적절한 리워드 금액을 추천합니다.

목표 참여 인원 달성을 위한 리워드 가격 설정

감이 아닌 데이터 기반 의사 결정
